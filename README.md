# Lululemon Stock Price Trend Analysis
A Python Data Analysis Project using Alpha Vantage API

## Project Overview
This project analyzes historical stock price data for **Lululemon (LULU)**.
It includes data fetching, cleaning, return calculation, moving average analysis, volatility measurement, visualization, and result export.

## Data Source
- API: Alpha Vantage
- Ticker: LULU (Lululemon Athletica Inc.)
- Data includes: Open, High, Low, Close, Volume, Daily Return, MA20, MA50, Volatility20, Cumulative Return

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## File Structure
- `Lululemon_Stock_Analysis.ipynb` — Main analysis code
- `Lululemon_Stock_Analysis_Result_AlphaVantage.csv` — Processed data results
- `Reflection_Report.pdf` — Project reflection report
- `demo.mp4` — Project demo video
- `README.md` — Project documentation

## Key Features
- Fetch real-time stock data via API
- Data cleaning and formatting
- Daily return & cumulative return calculation
- 20-day / 50-day moving average
- 20-day annualized volatility
- Price trend & volatility visualization
- Export cleaned data to CSV

## How to Run
1. Open `Lululemon_Stock_Analysis.ipynb`
2. Install required packages
3. Run all cells
4. View charts and output CSV
